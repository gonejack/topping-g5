Important note:

1,This firmware upgrade is available only on MAC systems and supports macOS 10.12 and above.
2, Never turn off or pull out power/USB cable during the update.



Firmware upgrade steps

1, Connect the product to your PC and turn on.

2, Run the exe file of this folder.

3, Click Browse.. and select the bin file of thie folder.

4, Click Start and wait for "Firmware upgrade finished successfully".

5, Check current firmware is already changed and click Exit and finish.

简体中文

注意：

1、这个固件升级包只能在苹果电脑系统上使用，并支持macOS 10.12及以上的系统。
2、升级的过程中请不要断开机子电源。



固件升级步骤：

1、将机子通过USB DAC口连接到电脑，并将机子开启

2、运行文件夹中的应用程序

3、点击Browse...选择文件夹中的bin文件

4、点击Start，等待几秒钟，提示"Firmware upgrade finished successfully."

5、观察Current Firmware ：已更改，即升级成功，点击"Exit"退出完成
